# Retail Analytics Case Study

**Tools:** Python | Tableau  
**Skills:** Exploratory Data Analysis | Customer Segmentation | 
Predictive Modelling

---

## Project Overview

This project analyses customer, order and product data from an 
Australian online activewear retailer to uncover sales patterns 
and support data-driven decision making.

Dataset: 5,090 customers and 6,177 orders (October 2023 - October 2024)

## Key Findings

- Sales peaked in October and June; summer months underperformed
- Google, Direct and Facebook were the strongest marketing channels
- ROOSTER5 and ROOSTER15 were the most effective coupon codes
- 65.5% of all orders included a discount
- Repeat buyers spend noticeably more than first-time buyers

---

## Predictive Models

| Model | Target | Performance |
|---|---|---|
| Linear Regression | Predict order value | R² = 0.999, MAE = 0.13 |
| Logistic Regression | Predict repeat purchase | Accuracy = 89%, AUC = 0.90 |
